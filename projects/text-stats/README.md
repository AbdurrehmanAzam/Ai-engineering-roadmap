# Text Stats CLI

A tiny command-line project that reads a text file and prints simple statistics:
lines, words, characters, and the most common words.

## Files
- `text_stats.py` — the CLI script.
- `sample.txt` — sample input to try the script.

## How to run
```bash
python text_stats.py sample.txt
```

## Sample output
```
File: sample.txt
Lines: 3
Words: 13
Characters: 81
Top words:
  ai: 2
  engineering: 2
  and: 1
  blends: 1
  build: 1
```
