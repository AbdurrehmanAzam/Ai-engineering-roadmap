# Phase 1.1 — Python Fundamentals

## What I Learned

- Running Python scripts from the terminal
- `print()`, `input()`, variables, and basic arithmetic
- Type conversion (`int()`, `float()`, `str()`)
- f-strings vs string concatenation
- Snake_case naming convention
- `git add` → `git commit` → `git push` workflow

## Files

| File | Description |
| :--- | :--- |
| `hello_world.py` | First Python program — prints "Hello, World!" |
| `add_two_numbers.py` | Adds two hardcoded numbers |
| `taking_input_from_user.py` | Interactive input with `input()` and type conversion |
| `input_two_numbers_sum.py` | Reads two numbers from the user and prints their sum |
| `area_of_square.py` | Computes the area of a square using `side²` |

## How to Run

```bash
# From the repo root
python phase-1-python/1.1-fundamentals/hello_world.py
python phase-1-python/1.1-fundamentals/area_of_square.py
```
