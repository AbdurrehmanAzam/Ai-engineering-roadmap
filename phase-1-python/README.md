# Phase 1: Programming Foundations (Python)

## Goals
- Build solid Python fundamentals (syntax, control flow, functions).
- Practice basic OOP concepts.
- Introduce NumPy and Matplotlib for numerical work.
- Learn virtual environments, Jupyter, and Git/GitHub basics.

## Key folders
- `1.1-fundamentals/` — small scripts for inputs, conditionals, strings, and lists.
- `1.2-oop/` — class and object exercises.
- `1.3-numpy-matplotlib/` — arrays and plotting practice.
- `1.4-venv-jupyter/` — environment setup notes.
- `1.5-git-github/` — Git basics and practice.

## How to run
1. (Optional) create and activate a virtual environment.
2. Run any script directly, for example:
   ```bash
   python 1.1-fundamentals/hello_world.py
   ```
3. Some scripts require interactive input in the terminal.
