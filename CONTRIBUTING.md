# Contributing

Thanks for your interest in this repository!

This is a **personal learning roadmap** — the code, notes, and logs here are part of my own AI Engineering journey. That said, you are very welcome to:

- ⭐ **Star** the repo if you find it useful or inspiring
- 🐛 **Open an issue** if you spot a factual error, a broken link, or a typo
- 💡 **Suggest resources** via an issue — especially free, open-source ones that fit the curriculum

## What I'm not accepting (right now)

- Pull requests that change the roadmap structure or curriculum order — the sequence is intentional
- Large refactors of existing code — this is a learning diary, not a production library

## Opening an issue

1. Check that a similar issue doesn't already exist.
2. Use a clear title (e.g. `[Phase 1] Broken link in 1.2-oop README`).
3. Describe the problem and, if applicable, what you expected instead.

## Code of Conduct

Please read and follow the [Code of Conduct](CODE_OF_CONDUCT.md) when interacting in this repository.
